# ZEN Course Hub

ZEN大学生向けの非公式履修管理ツールです。

履修予定・履修済み科目を管理しながら、卒業要件の達成状況を簡単に確認できます。

##公開URL：https://zencoursehub.netlify.app/

## 主な機能

- 授業検索（授業名・教員名）
- 学年・学期・分野・必修区分によるフィルタ
- 履修予定 / 履修済みの管理
- 取得単位・履修予定単位の自動集計
- 卒業要件分析
  - 総単位
  - 導入科目
  - 基礎科目
  - 展開科目
  - 基盤リテラシー
  - 多言語情報理解科目
  - 世界理解科目
  - 卒業プロジェクト科目
- データはブラウザの LocalStorage に保存

## コンセプト

「1画面で履修計画が完結すること」

履修登録前に Portal と見比べながら素早く計画を立てられることを目的としています。

## 注意事項

本ツールは非公式です。

卒業要件判定・履修判定・単位計算について正確性を保証するものではありません。

履修登録および卒業要件の最終確認は、必ず大学公式の学生要覧・シラバス・Portal等をご確認ください。

本ツールの利用によって生じた損害について、開発者は責任を負いません。
